# Git 명령어

{% gist https://gist.github.com/rbfl6418/4be87eb7760e79da58442eb49a96d6ac.js gist.md %}


### Commit하기

전체 track 하기

`git add.`

커밋 하기 

`git commit -m "메시지";`

### 데이터 조회하기

git에 올라가 있는 폴더 및 파일 확인(조회)

`git ls-files`

### 데이터 갱신하기

git repository pull받기

`git pull origin main`

git push하기

`git push origin main`

### 데이터 삭제하기

원격 저장소 & 로컬 저장소  둘 다 삭제

- 폴더 삭제

    `git rm -r 폴더명`

- 파일 삭제

    `git rm 파일명`

원격 저장소만 삭제

- 폴더

    `git rm --cached -r 폴더명`

- 파일

    `git rm --cached 폴더명`
